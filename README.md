# CRUD IU for students database

This project was made as homework project for the database course during my fourth semester on the UTM

The main point of this project is to improve my management of databases from python while I try to make things stylish, just for myself.

I've decided to integrate Plotly not just for beauty, but to learn another libraries from python that can make my portfolio better. I think thats one thing that defines me as a person.

I've already done some projects with pyqt6, some are'nt in my github rep, I'll upload them later.

Enjoy this integrated project.


This project uses these libraries:
    PyQt6: for beauty UI.
    Pydantic: for data verification.
    Plotly: To show statistics.
    PsyCopg2: To enter our docker postgreSQL container.
